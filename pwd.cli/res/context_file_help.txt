Commands:

<Enter>
   prints the current file with obscured passwords
..
   closes the current file
.cc field
   copies value of the field into the clipboard
.clear
   clears the screen
.ccu
   shortcut for `.cc user`
.ccp
   shortcut for `.cc password`
.edit [editor]
   opens the current file in the editor
.help
   prints this help
.lock
   locks the application
.pwd
   generates passwords
.rename name
   renames the opened file
.rm
   moves the current file to the folder `.bin`
.unobscured
   prints the file with passwords
