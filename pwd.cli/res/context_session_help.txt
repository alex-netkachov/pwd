To list the files in the folder just press <Enter>. Type a few characters
and press <Enter> to view only the names that begin with the input. Press
<Tab> for iterating over the matches.

Commands:

.add file
   creates a file
.clear
   clears the screen
.export path
   exports files to a new repository with a new password
.help
   prints this help
.html path
    exports to a password-protected html page
.lock
   locks the application
.lock ( pin | password | none )
   changes auto-lock type
.open path
   opens file by path
.pwd
   generates passwords
